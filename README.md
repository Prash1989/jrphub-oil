# JRP Oils Website v1

Premium, responsive Phase 1 corporate product showcase for JRP Oils.

## Deploy
Upload all files and folders to a GitHub repository and enable GitHub Pages from the repository settings.

## Files
- index.html
- style.css
- script.js
- assets/images (local SVG artwork)
- assets/icons

## Before production
- Replace placeholder contact/location details.
- Connect the inquiry form to Brevo, Formspree, a backend API, or another form service.
- Replace placeholder certification references with officially approved certifications only.
- Add real social media URLs.
